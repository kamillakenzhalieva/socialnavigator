# socialnavigator
Социальный навигатор — это единая цифровая платформа и интерактивная карта, разработанная для навигации людей с ОВЗ и ментальными особенностями, а также их семей, в инфраструктуре Астраханской области.

📌 О проекте
Проект решает проблему отсутствия единой актуальной базы данных в регионе. Мы объединяем информацию о государственных учреждениях, НКО и частных центрах помощи в одном интерфейсе. Проект разработан совместно с Астраханской организацией помощи детям-инвалидам и их семьям.

Интерактивная карта: Визуализация доступных точек помощи на базе API Яндекс.Карт.

Умные фильтры: Поиск услуг по типу помощи, возрастам и географическому положению.

Экспертная база: Верифицированная информация, учитывающая реальные потребности пользователей региона.

🛠 Технологический стек
Backend: Java 17+, Spring Boot (Spring Web, Spring Data JPA).

Database: PostgreSQL.

Maps API: Yandex Maps JavaScript API.

Support Bot: Интегрированная бета-версия помощника.
